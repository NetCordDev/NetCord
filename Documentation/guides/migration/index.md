# Migration Guide Overview

> [!NOTE]
> This section helps developers transition from other Discord libraries to NetCord.

NetCord offers a modern, performant alternative to existing Discord bot libraries. This guide will help you understand the differences and migrate your existing bot.

## Choose Your Migration Path

Select the library you're migrating from:

### C# Libraries
- @"migration/discord-net" - Discord.Net
- @"migration/dsharpplus" - DSharpPlus
- @"migration/other-csharp" - Other C# Libraries

### Other Languages
- @"migration/other-languages" - JavaScript, Python, Java, and more

## Why NetCord?

- **Modern C#** - Built with latest .NET features
- **Performance** - Optimized for speed and memory efficiency
- **Flexibility** - Host, Services, or bare-bones approaches
- **Type Safety** - Strong typing throughout
- **Active Development** - Regular updates and improvements
